# 0x06. Unittests in JS

## Description
This project focuses on learning how to write effective unit tests in JavaScript using tools such as Mocha, Chai, and Sinon. It also covers integration testing with a small Node.js server and various techniques for testing asynchronous functions.

## Learning Objectives
- How to use Mocha to write a test suite.
- How to use different assertion libraries (Node or Chai).
- How to present long test suites.
- When and how to use spies.
- When and how to use stubs.
- What are hooks and when to use them.
- Unit testing with asynchronous functions.
- How to write integration tests with a small Node.js server.

## Requirements
- **Environment**: Ubuntu 18.04, Node.js 12.x.x
- **Editors**: vi, vim, emacs, Visual Studio Code
- **File format**: `.js` extension
- **Execution**: All tests should run using `npm run test *.test.js` without warnings or errors.

## Tools and Technologies
- **Testing Framework**: Mocha
- **Assertion Libraries**: Node.js assertions, Chai
